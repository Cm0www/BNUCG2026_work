# 贝塞尔曲线与 B 样条曲线实现

---

## 项目介绍
本项目基于 **Python + Taichi** 实现参数曲线的交互式绘制与渲染，包括：**贝塞尔曲线、De Casteljau 算法、GPU 加速光栅化、返走样（抗锯齿）、均匀三次 B 样条曲线**。

通过本实验，深入理解参数曲线的几何意义、生成算法、光栅化原理、GPU 渲染流程以及交互系统实现。

---

## 实现功能
### 1. 贝塞尔曲线
- 使用 **De Casteljau 递归算法** 计算曲线上点
- 支持任意数量控制点（≥2 个）
- 实时生成平滑曲线
- 鼠标左键添加控制点
- C 键清空画布
- GPU 并行渲染，保证流畅运行

### 2. 光栅化渲染
- 操作 800×800 像素缓冲区
- 归一化坐标 → 屏幕像素坐标映射
- 边界安全检查，防止越界写入
- CPU 批量计算 → 一次性传输 GPU
- 高性能渲染（1000 采样点）

### 3. 返走样（抗锯齿）
- 3×3 邻域加权渲染
- 保留亚像素精度
- 高斯距离衰减权重
- 曲线边缘平滑无锯齿
- 显著提升视觉效果

### 4. B 样条曲线
- 均匀三次 B 样条（矩阵形式实现）
- 局部控制特性
- 支持任意多个控制点（≥4 个）
- 按 B 键切换贝塞尔 / B 样条模式

## 实现原理
### 1. De Casteljau 算法
通过递归线性插值，从控制点序列逐层插值，最终得到曲线上的点。
核心公式：
```
P0' = (1 - t) * P0 + t * P1
```

### 2. 均匀三次 B 样条
使用固定基矩阵实现，每 4 个控制点生成一段曲线，保证局部控制。
```
M = [
    [-1,  3, -3, 1],
    [ 3, -6,  3, 0],
    [-3,  0,  3, 0],
    [ 1,  4,  1, 0]
] / 6.0
```

### 3. 返走样原理
对浮点坐标附近 3×3 像素根据距离分配权重，距离越近权重越大，实现平滑边缘。
